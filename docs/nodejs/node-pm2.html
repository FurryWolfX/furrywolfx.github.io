bash 判断 ​

如果已经存在就重启，不存在就创建

bash
name="flow" ## 项目名
info=`pm2 ls | grep $name` ## 执行过滤并将结果赋值

## 在info中查找子字符串name
if [[ "$info" == *"$name"* ]]; then
  echo "restarting..."
  pm2 restart $name
else
  echo "initing..."
  pm2 --name $name start npm -- run start
fi
