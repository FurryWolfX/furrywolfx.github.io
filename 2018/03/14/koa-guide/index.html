Koa 服务搭建笔记

2018-03-15
  • JavaScript
  • NodeJS

koa 是由 Express 原班人马打造的，致力于成为一个更小、更富有表现力、更健壮的 Web 框架。 使用 koa 编写 web 应用，通过组合不同的 generator，可以免除重复繁琐的回调函数嵌套， 并极大地提升错误处理的效率。 koa 不在内核方法中绑定任何中间件， 它仅仅提供了一个轻量优雅的函数库，使得编写 Web 应用变得得心应手。

# 0x01 安装 Koa

yarn add koa
1

# 0x02 编写一个入口文件 app.js

const Koa = require("koa");
const app = new Koa();

// response

  ctx.body = "Hello Koa";

app.listen(3000);

# 0x03 搭建 Router 与 Controller

Koa 本质上是调用一系列的中间件，来处理对应的请求，并决定是否传递到下一个中间件去处理。我们来写一个最简单的中间件试试。

app.use(ctx => {
  ctx.body = `您的网址路径为：${ctx.request.url}`;
});
1
2
3

这段代码中，app.use 的 function 就是最简单的一个中间件，接受了请求，读出请求路径，并返回到客户端。

所以，接口的本质，就是判断不同的请求链接，干不同的事情，返回相应的结果。那么我们得需要一个路由中间件来处理分发请求。

开源的时代，当然是拿来主义了。github 搜下 koa-router (opens new window)，成功找到。根据它的介绍，我们先在项目根目录执行

yarn add koa-router
1

加入以下代码：

const Router = require("koa-router");
const router = new Router();

router.get("/", ctx => {
  ctx.body = `这是主页`;
});

router.get("/user", ctx => {
  ctx.body = `这是 user 页`;
});

// ...

app.use(router.routes()).use(router.allowedMethods());
app.listen(3000);

# 0x0A 自定义数据库中间件

参考文章： https://www.cnblogs.com/pqjwyn/p/6580472.html (opens new window)

每个中间件都是 generator 函数，当一个请求过来的时候，会依次经过各个中间件进行处理，当遇到 await next() 时，Koa 暂停了该中间件，执行下一个中间件。 直到某个中间件不调用下个中间件，即没有处理 await next()。则 koa 会逆序向上依次执行被暂停的中间件的剩余部分逻辑。

我们先安装 orm

yarn add orm
1

原生的使用方式可以查看文档 https://github.com/dresende/node-orm2 (opens new window)

现在我们来把它做成 Koa 中间件。

编写文件 orm-middleware.js 如下：

/**
 * 自己写的 orm 中间件
 */

const orm = require("orm"); // 使用说明 https://github.com/dresende/node-orm2

let connection = null;

function setup(db) {
  require("./models/demoModel")(orm, db);
}

module.exports = async function(ctx, next) {
  if (connection) {
    await next();
  } else {
    await orm.connectAsync("mysql://root:junlian@192.168.1.22:3306/zentao").then(function(db) {
      connection = db;
      /**
       * 包装 Promise 版本的 execQuery
       */
      db.driver.execQueryAsync = function(sql) {
        return new Promise((resolve, reject) => {
          ctx.req.db.driver.execQuery(sql, function(err, data) {
            if (err) {
              reject(err);
            } else {
              resolve(data);
            }
          });
        });
      };
      /**
       * 载入模型
       */
      setup(db);
      ctx.req.models = db.models;
      ctx.req.db = db;
    });
    await next();
  }
};
