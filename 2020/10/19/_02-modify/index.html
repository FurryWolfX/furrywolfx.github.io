# 拆边修改器

# 表面细分修改器

用于圆滑物体，可以配合折痕使用，标记不需要圆滑的边

# 多级细分修改器

和表面细分类似，但一般配合雕刻使用

可以设置合并距离

# 遮罩修改器

配合顶点组使用，没有加入顶点组的会被隐藏

# 实体化修改器
