Linux 中 pip 安装步骤与使用详解

2016-09-26
  • Linux
  • Python

# pip 下载安装

# pip 下载

代码如下

# wget "https://pypi.python.org/packages/source/p/pip/pip-1.5.4.tar.gz#md5=834b2904f92d46aaa333267fb1c922bb" --no-check-certificate
1

# pip 安装

代码如下

# tar -xzvf pip-1.5.4.tar.gz
# cd pip-1.5.4
# python setup.py install
